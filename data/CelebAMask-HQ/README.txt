File Information
========================

- High-Resolution Face Images (./CelebA-HQ-img)

- CelebAMask-HQ to CelebA Image ID Mapping (./CelebA-HQ-to-CelebA-mapping.txt)

- Mask Annotations (./CelebAMask-HQ-mask-anno)

- Pose Annotations (./CelebAMask-HQ-pose-anno.txt)

- Attributes Annotations (./CelebAMask-HQ-attribute-anno.txt)

- Evaluation Partitions (We use the same train/val/test split as the CelebA dataset)
